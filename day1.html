The src attribute

HTML images are defined with 'img' tag, and the filename of the image source is specified in the src attribute.:
There are two ways to specify URL in src attribute:
1. Absolute URL or external url, which is you cannot control due to may be violation of copyright. Example src='https://www.w3schools.com/images/img_girl.jpg'.
2.Relative URL- links of images which are hosted within the website, doesnot contain domain name. Example src='imge_girl.jpg'.
Tip: It's almost always best to use relative URL. They will not break if you change domain.

The width and height attrbutes

The width and height of the 'img' tag defines the width and of the image.

The alt attributre

The alt attribute should reflect the image content, so user who cannot see the image get an understanding what the image contains.

The HTML attributes/link

HTML links are defined with the 'a' tag. The link address is specfied in the href attribute.
